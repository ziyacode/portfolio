# Ziya Səfərzadə – Portfolio

Bu qovluqda Ziya Səfərzadənin şəxsi portfolio saytı yerləşir. Sayt statik HTML, CSS və JavaScript istifadə etməklə hazırlanıb.

## Fayllar

- `index.html` – əsas səhifə və bütün məzmun
- `styles.css` – ümumi dizayn, responsive quruluş və animasiyalar
- `script.js` – scroll animasiyaları, naviqasiya və illərin avtomatik yenilənməsi
- `package.json` – sadə local preview üçün skriptlər

## Lokal işə salınma

Terminalda bu qovluğa keçin və:

```bash
npm install
npm run start
```

Əgər `npm` istifadə etmək istəmirsinizsə, sadəcə `index.html` faylını brauzerdə açmaq kifayətdir.
